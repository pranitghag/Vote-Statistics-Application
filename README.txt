Run result.py to run the application.

result.py file has class Result that inherits from class Election from file election.py
election.py is a custom library
myLib.py is a custom library that imports an eternal library matplotlib to plot the datapoints.
myLib.py has a constant that is used by the main file.

The data structure used in this project is a dictionary. Dictionaries have key value pairs with data stored that can be extracted
using the key.

The files included will be dataset.py to create a dataset, the dataset with 50,000 lines to test the project, myLib.py, election.py
and results.py.

After running the applciation output is printed on the console, graphs are plotted and a results.txt file is formed with data if it
has to be used for further analysis and planning.
